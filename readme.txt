Steps to execute this program :

1) Download NodeJS and install to your computer : https://nodejs.org/en/download/
2) Open cmd of Trace with Us file 
3) type npm install
4) Go to src/database file
5) import trace_with_us sql file to your xampp
6) back to cmd of Trace with Us file 
7) type npm start to execute
8) Go to your browser 
9) search for http://localhost:3000
